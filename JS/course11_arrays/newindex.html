У вас есть сервис по доставке еды

Если у клиента есть промокод, то показавать ему список блюд с 10% скидкой

Если у клиента нет промокода, то показавать ему список блюд без скидки

Если в меню есть хотя бы одно блюдо со стоимостью больше 5000тг, то перевернуть массив наоборот

Посчитать сколько в меню блюд, названия которых начинается на "К"
